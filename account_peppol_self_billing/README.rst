This module adds support for **Peppol Self-Billing validation** for
suppliers.

It allows companies using Self-Billing to ensure that their suppliers
explicitly support **Peppol Self-Billing**, while exchanging invoices
through the Peppol network.

The module performs an additional validation step to verify that the
supplier declares support for Peppol Self-Billing via the Peppol
network.
